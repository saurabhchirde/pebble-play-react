## Features

- User Authorization
  - Sign In
  - Sign Up
  - Sign Out
- Protected/Private Routes
- Video Listing
- Filter by Category
- Search Videos
- Liked Videos
- Watch Laters
- Watched History
- Create and Delete Playlist
- Single Video View
- User Account Page

</br>

## Tech Stack

**Client:** React, React Router, Redux, Context API, [Pebble UI](https://pebbleui.com/)

**Server:** Mockbee (Mock Backend)

</br>

## Run Locally

Clone the project

```bash
  git clone https://github.com/saurabhchirde/pebble-play-react.git
```

Go to the project directory

```bash
  cd pebble-play-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
